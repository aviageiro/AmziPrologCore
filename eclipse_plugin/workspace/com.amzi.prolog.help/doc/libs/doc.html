Prolog Libraries

The documentation for each of the libraries is in the source code.

  • List Predicates -- For manipulating lists.
  • Date & Time Predicates -- Provide date and time logical arithmetic.
  • XML Parsing -- Converts between Prolog structures and XML equivalent.
  • Miscellaneous Predicates -- Includes freeze, melt, string comparison, and flag handling.

On our website you will find customer-contributed libraries, including a SICStus/SWI Compatibility Library by John Cullen.

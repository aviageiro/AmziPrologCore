    Purpose

    The Overview and Programmer's Guide provide a conceptual framework for using all aspects of the Logic Server. If you are impatient and want to get started right away, go to the 'Hello Prolog' section in the reference document for the environment you are working in.

    This document assumes you have a reasonable working knowledge of Prolog.

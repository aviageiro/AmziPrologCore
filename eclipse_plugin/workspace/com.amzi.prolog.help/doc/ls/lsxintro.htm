LSX Introduction

The Logic Server includes a number of toolkits, described in this document.

ODBC Support - An LSX that implements ODBC connectivity from Prolog. It can be used from the IDE as well as applications written in any Windows development environment.

Sockets Support - An LSX that implements the Sockets protocol for implementing clients and servers in Prolog.

O.S. Specific Predicates - An LSX containing many operating-system dependent extended predicates.

Delphi Graphics Predicates - A collection of extended predicates that give Prolog direct access to Delphi forms and controls.

Command-Line Runtime Shell - The source code for the Windows console runtime. It is a relatively simple Logic Server application, that runs an .xpl file.

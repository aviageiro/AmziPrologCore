Visual Basic Samples

These samples illustrate the use of the Visual Basic and the Amzi! Logic Server.

ExtPred - This example shows how to create Prolog predicates that are implemented by Visual Basic procedures using VB 5.0 or later.

Hello - Hello Prolog from Visual Basic.

Types - A VB program that exercises a number of features of the Logic Server API.

WGene - The Genealogy application implemented with a VB front-end. The people data and the rules are both stored in Prolog.

VB.NET Pets Sample

This sample demonstrates calling a simple expert system from VB.NET.

Building the Sample

  1. Open the pets project.
  2. Make sure the 'References' include amzinet.dll.
  3. Build pets.exe.
  4. Optionally copy amzi.dll to the bin\Debug subdirectory where amzinet.dll and pets.exe are located.
  5. Compile and link pets.pro and copy pets.xpl to the bin\Debug subdirectory.

Running the Sample

  1. Run pets.exe.
  2. Enter a sound like 'woof', 'quack' or 'meow'.

Delphi Hello Sample

This is the simplist Delphi sample. It ensures everything is installed correctly.

Building the Sample

  1. Compile and link hello.pro into hello.xpl.
  2. Open the Delphi Project hellop.dpr.
  3. Build eq.exe.

Running the Sample

  1. Open the Delphi Project hellp.dpr and run it, or double-click on hellop.exe that you built above.

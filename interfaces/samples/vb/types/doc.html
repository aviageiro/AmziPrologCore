Types -- Visual Basic Types Demonstration Sample

This sample demonstrates how to pass the various Prolog types back and forth between VB and Prolog..

Building the Sample

To build it:

  1. Compile testvb.pro
  2. Link testvb.plm into testvb.xpl.
  3. Open the VB project testvb and build testvb.exe.

Running the Sample

To run it:

  1. Run testvb.exe.

Java Samples

These samples illustrate the use of the Amzi! Logic Server from Java.

Hello - Hello Prolog from Java, used to see if you've got all of the Java stuff installed correctly.

Pets_Callback - The Pets expert system, illustrating multiple engines and extended predicates in a console Java application.

Ship - A prototype expert system for recommending package shipping options based on package weight, destination and priority. It uses a GUI developed in Java.

Hello Load URL - The Hello sample modified to load hello.xpl from a JAR file from a URL and also demonstrates loading the xpl file from a byte array.

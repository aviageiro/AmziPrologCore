C++ Pets COM Sample

This sample demonstrates calling a simple expert system from C++. It imports the type library amzicom.tlb and calls the amzicom.dll which in turn calls amzi.dll.

Building the Sample

  1. Register the Amzi! COM Logic Server by typing regsvr32 amzicom.dll from the amzi/bin directory.
  2. Compile and link pets.pro into pets.xpl.
  3. Under Windows, open the Microsoft Visual C++ Project cpp_pets.dsw and build cpp_pets.exe.
  4. Copy pets.xpl to the Debug subdirectory.

Running the Sample

  1. Run cpp_pets.exe.

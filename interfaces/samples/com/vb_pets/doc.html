Visual Basic Pets COM Sample

This sample demonstrates calling a simple expert system from Visual Basic. It references amzicom library which calls amzicom.dll which in turn calls amzi.dll.

Building the Sample

  1. Register the Amzi! COM Logic Server by typing regsvr32 amzicom.dll from the amzi/bin directory.
  2. Compile and link pets.pro into pets.xpl.
  3. Under Windows, open the Microsoft Visual Basic Project vb_pets.vbp and build vb_pets.exe.

Running the Sample

  1. Run vb_pets.exe.
  2. Highlight a sound and press the button 'Get Pet'.

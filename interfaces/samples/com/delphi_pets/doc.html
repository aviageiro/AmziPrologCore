Delphi Pets COM Sample

This sample demonstrates calling a simple expert system from Delphi. It references amzicom library which calls amzicom.dll which in turn calls amzi.dll.

Building the Sample

  1. Register the Amzi! COM Logic Server by typing regsvr32 amzicom.dll from the amzi/bin directory.
  2. Compile and link pets.pro into pets.xpl.
  3. Open the Delphi Project delphi_pets.dpr.
  4. Import the amzicom.tlb type library into Delphi.
  5. Build delphi_pets.exe.

Running the Sample

  1. Run delphi_pets.exe.
  2. Highlight a sound and press the button 'Get Pet'.

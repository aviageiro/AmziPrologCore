Delphi LSAPI Sample

This sample demonstrates the a variety of the LSAPI calls from Delphi.

Building the Sample

  1. Compile test.pro and link it into test.xpl.
  2. Open the Delphi Project testp.dpr.
  3. Build testp.exe.

Running the Sample

  1. Open the Delphi Project testp.dpr and run it, or double-click on testp.exe that you built above.

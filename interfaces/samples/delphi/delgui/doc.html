Delphi GUI Sample

This sample demonstrates the use of extended predicates to build a graphical user interface in Prolog. The file delgui.pro has a small set of reusable graphics predicates.

Building the Sample

  1. Build the dgtemp.ppj project, or compile dgtemp.pro and delgui.pro and link them into dgtemp.xpl.
  2. Open the Delphi Project dgtemp.dpr.
  3. Build dgtemp.exe.

Running the Sample

  1. Open the Delphi Project dgtemp.dpr and run it, or double-click on dgtemp.exe that you built above.

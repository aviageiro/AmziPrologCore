Hello -- Visual Basic Hello Sample

This is the first Visual Basic sample to get running. It ensures you have everything installed correctly.

Building the Sample

To build it:

  1. Compile hello.pro
  2. Link hello.plm into hello.xpl.
  3. Open the VB project hello and build hello.exe.

Running the Sample

To run it:

  1. Run hello.exe.

Java Hello Sample

This is the simplist Java sample. It ensures everything is installed correctly.

Building the Sample

  1. Compile and link hello.pro into hello.xpl.
  2. Compile Hello.java with javac.

Running the Sample

  1. Type java Hello to run the program.
